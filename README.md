## ✨ Fitur

- **Dark Cyber Theme** — estetik gelap dengan aksen teal `#00e5c3`
- **Custom Cursor** — dot + ring yang mengikuti gerakan mouse
- **Parallax Hero Glow** — efek cahaya yang bergerak sesuai posisi kursor
- **Scroll Reveal** — elemen muncul saat di-scroll menggunakan IntersectionObserver
- **Language Bar Animation** — progress bar bahasa yang animate saat terlihat
- **Noise Texture Overlay** — detail tekstur halus untuk kedalaman visual
- **Responsive Design** — tampil baik di desktop, tablet, dan mobile
- **Zero Dependencies** — murni HTML, CSS, dan Vanilla JavaScript

## 🚀 Cara Pakai

### 1. Clone repo

```bash
git clone https://github.com/AdanaTatuyaHatsumi/portfolio-dark-tency.git
cd portfolio-dark-tency
```

### 2. Buka di browser

Cukup buka file `index.html` langsung di browser, tidak butuh server.

```bash
# Opsional: jalankan dengan live server (VS Code extension)
# atau pakai Python HTTP server
python -m http.server 8000
```

### 3. Kustomisasi

Edit bagian berikut di `index.html` sesuai kebutuhan:

| Bagian | Lokasi di file |
|--------|---------------|
| Nama & role | Section `#hero` |
| Bio & info | Section `#about` |
| Daftar skill | Section `#skills` |
| Proyek & pengalaman | Section `#experience` |
| Pendidikan | Section `#education` |
| Social links | Section `#contact` |

Untuk mengubah warna tema, edit CSS variables di `style.css`:

```css
:root {
  --bg:     #0a0a0f;   /* Background utama */
  --accent: #00e5c3;   /* Warna aksen (teal) */
  --text:   #e8e8f0;   /* Warna teks utama  */
  --muted:  #6b6b80;   /* Warna teks redup  */
  --card:   #13131c;   /* Background kartu  */
}
```

## 🌐 Deploy ke GitHub Pages

1. Push repo ke GitHub
2. Buka **Settings → Pages**
3. Set source ke branch `main`, folder `/ (root)`
4. Akses di `https://username.github.io/portfolio`
